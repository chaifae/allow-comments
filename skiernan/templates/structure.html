HTML (HyperText Markup Language) is the structure of a webpage. HTML is the standard markup language used to create webpages.

DOM (Document Object Model) refers to the tree-like structure of a webpage. DOM is a cross-platform and language-independent convention for representing and interacting with objects in markup languages. The nodes of every document are organized in a tree structure, called the DOM tree.

Websites are all made up of boxes, with boxes inside those boxes, and more boxes inside those. Box-ception. When reading the HTML you can see the "boxes" by the indentation in the code - the further indented the code is the deeper the visual box is nested.
