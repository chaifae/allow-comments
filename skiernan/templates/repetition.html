If statements let us control what part of the code is executed. If the condition is True, the indented block of code will execute. If not the interpreter moves on without executed the block under the if statement. An else statement under an if statement is executed when the if statement was not True.

While statements will loop through the indented block of code until the condition is no longer met. Both while and if statements use comparison operators, like greater than or less than, to return the Boolean values True and False.

Lists are sequences of data contained in square brackets [ ] and can contain any combination of numbers, strings, or Boolean values. Lists can even contain other lists. For loops allow you to iterate over lists.
